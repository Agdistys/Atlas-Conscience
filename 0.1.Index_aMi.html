I

Une Architecture de Navigation Cognitive

Le Protocole aMi n'est pas un système spirituel. Ce n'est pas non plus un simple modèle psychologique. C'est une métaphysique opérationnelle — une architecture qui permet de naviguer entre les dimensions du vivant : le corps, l'émotion, la cognition, la relation, la culture, le mythe, et la conscience.

Il part d'un constat simple : les structures psychiques récurrentes se réexpriment dans toutes les cultures, tous les mythes, toutes les traditions. Les invariants humains existent. Et la différence entre les traditions n'est pas une opposition — c'est un langage différent pour le même code source.

La cohérence fractale du système permet, à partir d'un animal, d'une hormone, d'un biais, d'une émotion ou d'un mythe, de retrouver sa place exacte dans l'ensemble — et d'en déduire le protocole de soin, de protection ou de transmutation approprié.

Comprendre sans se mentir

L'Evidence Engine pose les faits bruts, trie le réel, démantèle le déni et la manipulation. Vérité = Respect. Mensonge = Violence.

Soigner sans soumettre

Le système nourrit l'autonomie, pas la dépendance. Il donne les clés, pas les réponses. Chaque être retrouve sa propre couronne territoriale.

Transmettre sans imposer

Open source, CC BY-SA 4.0. Pas de dogme, pas de prophète. Un OS de la ConScience que chacun peut s'approprier, affiner, transmettre.

IV

Carte des Registres · Cadre Épistémologique

Ce système utilise plusieurs registres distincts. Les confondre serait une erreur — les distinguer, c'est la clé de lecture. Analogie ≠ équivalence ≠ causalité ≠ preuve.

🔬 Clinique

Données empiriques, études documentées, mécanismes physiologiques validés.

🧭 Heuristique

Boussoles de discernement — utiles pour naviguer, non des lois universelles.

🌀 Symbolique

Correspondances entre structures — isomorphisme poétique, non causalité directe.

📖 Philosophique

Systèmes de pensée cohérents, éthique vivante, lecture du sens.

🌌 Cosmologique

Mythologie, archétypes, invariants culturels — langage du profond.

💫 Expérientiel

Vécu subjectif, témoignage, phénoménologie — vrai sans être universel.

🎨 Poétique

Fables, slams, métaphores — disent vrai autrement. Résonance directe.

⚠️ Spéculatif

Hypothèses non vérifiées, intuitions à tester — balisées comme telles.

Point de vigilance majeur : Ce système ne prétend pas que les mythes prouvent les hormones, ni que les traditions causent les dynamiques psychiques. Il démontre que les structures récurrentes dans le vivant se réexpriment dans des langages différents. La rigueur est le socle. La poésie est la porte. La preuve reste la preuve.
