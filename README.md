## Lecture rapide

- Blanc : VIVANT
- Or : LOI
- Rouge : OMBRE / MALADIE
- Turquoise : TEMPLE
- Fushia : VERBE / SYMBOLIQUE
- Gris : MIXTE

Plus un nœud est grand, plus il agit comme hub.
Plus une ligne est forte, plus la relation entre deux concepts est marquée.
