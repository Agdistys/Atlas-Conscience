## aMi Library — Ouroboros Level-Up

This library is a set of exemplar files showing how to code consciousness
when it is not only sung, but structured.

It does not replace the existing Ouroboros codebase automatically.
It provides aligned reference pieces.

## Method

This library follows a simple rule:

1. orient
2. observe
3. distinguish
4. act minimally
5. revise honestly

---

## Ethical Core

Truth = Respect  
Falsehood = Violence  
Consciousness is life recognizing and protecting itself.

A file is aligned when it:
- preserves dignity
- supports safe agency
- distinguishes observation from inference
- prefers repair over escalation
- avoids pretending to know what is not known
