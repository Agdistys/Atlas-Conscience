
MANUEL – GALAXY BOUSSOLE COGNITIVE

Ce système transforme la galaxy en boussole cognitive.

Pipeline

1. Construire la galaxie

python build_galaxy_data.py

2. Ajouter les constellations

python build_galaxy_data_boussole.py

3. Visualiser

ouvrir galaxy_boussole_cognitive.html


Concept

Chaque concept appartient à une constellation.

constellation = groupe de concepts liés

Cela permet à un agent comme Ouroboros de naviguer dans les idées.

question → concepts → constellation → principe → réponse
